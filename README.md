# Einmalige Einrichtung der Firebase (5 Minuten, kostenlos):
- Auf https://console.firebase.google.com ein Projekt anlegen (google-Account z.B. gmail nötig)
- „Realtime Database" aktivieren → Regeln auf „Test-Modus" (30 Tage offen)
- Die Datenbank-URL aus der Konsole kopieren (sieht aus wie https://xyz-default-rtdb.europe-west1.firebasedatabase.app)
- Diese URL oben in Lehrer und Schüler HTML-Dateien eintragen!
